GeoReporter for Android is a native Open311 GeoReport v2 reference client. If any city wishes to be included in the default list of available Open311 GeoReport v2 endpoints, please let us know though the open311-mobile listserv.

About Open311 GeoReport v2

Open311 is an open protocol for communicating non-emergency service requests to and from municipal governments using the GeoReport v2 specification. GeoReporter implements the GeoReport v2 specification, including some additional feaures that will be added to the next specification iteration soon. For additional information on GeoReport v2 specification visit: http://wiki.open311.org/GeoReport_v2

For more about Open311, visit http://open311.org/learn/ and http://lists.open311.org/groups/discuss.
